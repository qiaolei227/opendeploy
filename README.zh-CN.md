大部分 "AI 辅助工具" 现在长这样:

> **顾问**: 客户要加个 xxx 功能
>
> **AI**: 好,我帮你写 Python 插件
>
> **顾问**: ……但字段得先在系统里建,审批流也得改
>
> **AI**: 那个你自己去配

**开达想做的是: 这一整串,AI 全干。**

## 怎么开始

需要 Windows 10/11 + 一个 LLM API Key。

```bash
pnpm install
pnpm dev
```

v0.1 Alpha 会提供现成安装包。
