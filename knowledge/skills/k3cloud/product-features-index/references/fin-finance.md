# 财务管理(AR/AP/GL)标准功能字典

**骨架状态**:本字典初版覆盖主干。

---

## 模块分层

- **AR** 应收(客户往来)
- **AP** 应付(供应商往来)
- **GL** 总账 + 凭证
- **FA** 固定资产(本 skill 未覆盖)
- **CB** 现金管理(银行日记账)

---

## 常用标准功能

### 凭证生成
- **需求关键词**:"业务单据自动生成凭证"
- **标准功能**:**凭证模板** 配置——每种业务单据(销售出库 / 采购入库 / 收款 / 付款)可配置自动生成凭证的规则
- **启用路径**:总账 → 凭证模板管理
- **典型误判**:顾问写 Python 在 AfterSave 里生成凭证 — **标准有凭证模板,别重写**

### 多币别
- **需求关键词**:"美元订单 / 欧元采购"
- **标准功能**:**币别** 基础资料 + 汇率策略(期末 / 固定 / 即时)
- **启用路径**:基础资料 → 币别;财务 → 汇率策略

### 税金
- **需求关键词**:"开发票"、"税率"、"增值税"
- **标准功能**:**税种 / 税率** 基础资料 + 单据上自动计算税额
- **启用路径**:基础资料 → 税种

### 应收账龄
- **需求关键词**:"欠款天数分析"
- **标准功能**:**账龄分析表**(按区间统计)
- **启用路径**:应收管理 → 报表

### 应收预警
- **需求关键词**:"快到期的应收提醒"
- **标准功能**:应收账龄 + 催收提醒(部分版本)
- **启用路径**:应收管理 → 催收管理

### 核销(收款 ↔ 应收单)
- **需求关键词**:"收款和订单对应"、"冲销欠款"
- **标准功能**:**手工核销** / **自动核销**
- **启用路径**:应收管理 → 核销

### 结算(付款 ↔ 应付单)
- **需求关键词**:"付款对应采购单"
- **标准功能**:应付核销,和应收对称
- **启用路径**:应付管理 → 核销

### 月结 / 年结
- **需求关键词**:"关账"、"月底结账"
- **标准功能**:**结账** 操作(逐模块关账:存货 → 应收 → 应付 → 总账)
- **启用路径**:各模块 → 结账;**不能跳过顺序**

### 报表
- **需求关键词**:"资产负债表"、"利润表"、"现金流量表"
- **标准功能**:**账表定义**(公式驱动)→ 内置模板可直接用,也可自定义
- **启用路径**:总账 → 报表 → 账表定义

## 典型"以为要二开"

| 顾问描述 | 真实答案 |
|---|---|
| "业务单自动做账" | 凭证模板 |
| "应收欠款天数" | 账龄分析报表 |
| "把收款冲掉对应订单" | 核销功能 |
| "按币别汇算" | 币别 + 汇率策略 |
| "生成资产负债表" | 账表定义 + 内置模板 |
| "多维度费用分摊" | 费用分摊 / 辅助核算 |

---

## 辅助核算 / 核算项目

- **需求关键词**:"按部门核算利润"、"按项目分摊费用"
- **标准功能**:**辅助核算** 项目(部门 / 客户 / 供应商 / 自定义)——科目勾选辅助核算后凭证可按该维度统计
- **启用路径**:总账 → 科目 → 辅助核算设置
- **常见坑**:用户说"我要按项目看成本" → 不是二开,是辅助核算配置
