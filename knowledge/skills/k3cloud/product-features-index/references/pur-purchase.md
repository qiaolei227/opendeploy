# 采购管理(PUR)标准功能字典

**骨架状态**:本字典初版覆盖主流程,细节场景随社区贡献扩展。

---

## 主流程

采购申请单 → 采购订单 → 采购收料通知 → 采购入库单 → 采购发票 → 应付单

每步可独立启用 / 关闭,由 **采购管理系统参数** 控制。

---

## 常用标准功能

### 供应商信用管理
- **需求关键词**:"供应商也要额度"、"预付款对控"
- **标准功能**:应付模块的 **供应商信用额度** + **预付款管理**
- **启用路径**:应付管理 → 基础资料 → 供应商信用策略

### 价格对比 / 历史价
- **需求关键词**:"看最近一次采购价"、"供应商比价"
- **标准功能**:采购订单表体有 **最近采购价** 带出功能;**供应商物料价格单** 可维护不同供应商同物料的标准价
- **启用路径**:采购管理 → 基础资料 → 供应商物料价格单

### 采购审批
- **需求关键词**:"大额采购要审批"
- **标准功能**:审批流 + 条件表达式,同销售模块
- **启用路径**:系统服务 → 审批流设计器

### MRP 运算触发采购申请
- **需求关键词**:"低于安全库存自动触发采购"
- **标准功能**:MRP 模块(本 skill 未覆盖,属制造模块)会自动生成采购申请单
- **启用路径**:生产管理 → MRP 运算

---

## 典型"以为要二开"

| 顾问描述 | 真实答案 |
|---|---|
| "采购订单要和合同关联" | 标准有采购合同单据 + 下推到订单 |
| "供应商报价管理" | 供应商物料价格单 + 价格更新 |
| "不同物料走不同审批" | 审批流条件 |
| "自动按历史价带出" | 采购订单新建时默认带最近价 |
