# 库存管理(STK)标准功能字典

**骨架状态**:本字典初版覆盖主流程。

---

## 主流程

其他入库单 / 采购入库单 → 库存更新 → 其他出库单 / 销售出库单 / 生产领料单 → 库存消耗

---

## 常用标准功能

### 多组织库存
- **需求关键词**:"按组织隔离库存"
- **标准功能**:**库存组织** 划分,各组织独立核算库存数量和金额
- **启用路径**:基础资料 → 组织机构 → 库存组织

### 多仓库 / 多仓位
- **需求关键词**:"一个仓库下多个货位"、"货架号"
- **标准功能**:**仓库** → **仓位**(货位) 两级,物料可指定仓位
- **启用路径**:基础资料 → 仓库;基础资料 → 仓位

### 调拨
- **需求关键词**:"跨仓库 / 跨组织调货"
- **标准功能**:**直接调拨单**(同组织)/ **调拨申请单 → 调拨发货 → 调拨收货**(跨组织)
- **启用路径**:库存管理 → 调拨系列单据

### 盘点
- **需求关键词**:"盘库"、"盘盈盘亏"
- **标准功能**:**盘点方案** → **盘点单** → 自动生成盘盈入库 / 盘亏出库
- **启用路径**:库存管理 → 盘点管理

### 库存预警(上下限)
- **需求关键词**:"低于最低库存提醒"、"高于最高库存提醒"
- **标准功能**:物料档案 → 库存信息 → **最低安全库存 / 最高库存** + 库存预警查询
- **启用路径**:物料档案 + 库存管理 → 库存预警

### 即时库存查询
- **需求关键词**:"看当前库存"
- **标准功能**:**即时库存查询**(内置报表,支持多维度筛选)
- **启用路径**:库存管理 → 即时库存查询

### 库存核算
- **需求关键词**:"月底算库存成本"
- **标准功能**:**库存核算方案**(加权平均 / 先进先出 / 标准成本),月末关账生成成本凭证
- **启用路径**:存货核算 → 核算方案

## 典型"以为要二开"

| 顾问描述 | 真实答案 |
|---|---|
| "库存低于 X 提醒" | 物料安全库存 + 库存预警查询 |
| "按批次管理" | **批号管理** 启用项,物料档案勾选 |
| "按序列号管理" | **序列号管理** 启用项 |
| "按保质期管理" | **保质期管理** 启用项 |
| "按箱管理 / 辅助单位" | **辅助单位** 标准功能 |
| "盘点生成盈亏单" | 盘点标准流程自动生成 |

---

## 启用项注意

很多"库存特殊需求"对应的是物料档案上的**启用项**(批号 / 序列号 / 保质期 / 辅助单位 / 寄售 / 配送)。**启用后对应标准字段和流程就亮了**,不是二开。

**典型误判**:顾问想给物料加"批号"字段 → 其实勾启用项就有完整批号流程。
