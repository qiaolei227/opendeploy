## 临时话术(本字典完工前 agent 怎么说)

用户问 MFG 相关需求时:

> "MFG 模块 OpenDeploy 知识库还没整理完整(目前只有模块清单)。我能给你的:
>
> 1. K/3 Cloud 标准的 MFG 主流程是 [生产订单 → 工序 → 完工 → 成本],你的需求大概率落在某一步
> 2. 建议你先在金蝶官方帮助(`help.open.kingdee.com/dokuwiki_std`)按关键词查一下,确认是不是已有功能
> 3. 如果你能告诉我具体场景(例如'按订单倒冲领料'),我可以帮你判断是配置 / BOS / 插件层"

**不要硬编 MFG 的具体功能名 / 启用路径 / 字段名**——会害人。
